# Titan-Web-Components

titan 智控平台复用组件。<br/>
组件需通过 es6 module 形式引入，本身项目环境需按照[vue-boilerplate](https://github.com/deepberry/vue-boilerplate)进行配置。

## Get Started

### 1.Install

安装前请参考[DeepBerry 私有包安装](https://github.com/deepberry/FE-docs/blob/master/packages/DeepBerry%E7%A7%81%E6%9C%89%E5%8C%85%E5%AE%89%E8%A3%85.md)进行对应设置。

```shell
npm install @deepberry/titan-web-components
```

### 2.Import

#### 全部引入

```js
import TitanUI from "@deepberry/titan-web-components";
app.use(TitanUI);
```

#### 部分引入

```js
import RobotLive from "@deepberry/titan-web-components/src/components/RobotLive/Index.vue";
```

### International

```html
<TitanConfigProvider locale="enUs">
    <!-- components -->
</TitanConfigProvider>
```
